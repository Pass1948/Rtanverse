⚠️ 외부 에셋 경로 안내
해당 프로젝트는 용량 제한으로 인해 외부 에셋이 GitHub에 포함되어 있지 않습니다.
프로젝트 실행 시 일부 오브젝트나 UI가 나타나지 않을 수 있으니, 아래 링크를 통해 외부 에셋을 다운받아야 합니다.
